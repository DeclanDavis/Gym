C - Create a Gym Member
R - Retrieve a Gym Member Randomly, by their Name, or All the Gym Members
Results
U - Update a Gym Member
Enter the ID of the Gym Member you wish to update.
Then please fill in the new data in the manditory:
Title, First Name, Last Name, Email Address and Premium Membership inputs.
D - Delete a Gym Member


C - Create a Gym Class
R - Retrieve a Gym Class Randomly, by it's Class Name, or All the Gym Classes
Results
U - Update a Gym Class
Enter the ID of the Gym Class you wish to update.
Then please fill in the new data in the manditory:
Class Name, Class Day, Session Length (Hrs) and Price inputs.
D - Delete a Gym Class


C - Create an Enrollment (Member Class)
Enter the ID of the Gym Member
and the ID of the Gym Class
they wish to enroll on.
R - Retrieve an Enrollment (Member Class) Randomly, the Enrollments of a Gym Member by entering their User ID, or All the Enrollments (Member Classes) in the entire gym.
Results
U - Update an Enrollment (Member Class)
Enter the ID of the Enrollment (Member Class) you wish to update.
Then please fill in the new data in the manditory:
User ID and Class ID inputs.
D - Delete an Enrollment (Member Class)
